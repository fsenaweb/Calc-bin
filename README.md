## Build Setup

``` bash
# instalar o VueJs
$ npm install -g vue-cli

# instalar o Quasar 
# npm install -g quasar

# instalar dependencia
$ npm install

# carregar o app atraves de localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
